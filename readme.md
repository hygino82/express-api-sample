https://www.youtube.com/watch?v=QqHaDrsj8VA&list=PLsCs8YwnvP88lUF2Z5qRP28pXBh3ef9Rz

## Install packages
```bash 
npm install
```

## Run project in dev mode
```bash 
npm run dev
```

## Build project
```bash 
npm run build
```

## Run compiled project
```bash 
npm run start
```
